Built on

  • BLADE framework — provides the LogReaderRegistrar + VorpalLogReader MBean that runs on every cluster node.
  • WLS DomainRuntime MBean federation — the AdminServer reads remote MBeans without manual JMX URLs.
  • JAX-RS — three endpoints (servers, servers/{name}/logs, servers/{name}/logs/{path:.+}) backed by LogQueryAPI.
  • Vanilla HTML/JS — no SPA framework; the viewer is server-rendered + a small fetch-driven polling loop.

Reporting issues

File issues at the project tracker. Please include the WLS / OCCAS version, the server list (sanitized of hostnames if needed), and the AdminServer log lines around the time of the symptom.
