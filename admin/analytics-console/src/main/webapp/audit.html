Resource Audit

The analytics pipeline needs five WebLogic resources to function end-to-end: a JMS Server hosting a Distributed Queue and Connection Factory inside a JMS Module, and a JDBC Data Source pointing at the analytics MySQL instance. This page walks the DomainConfiguration MBean tree and reports what's wired.
