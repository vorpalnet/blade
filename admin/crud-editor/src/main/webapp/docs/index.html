Welcome

The CRUD Editor is a generic browser for BLADE service resource tables. Where the Configurator edits a service's configuration (one JSON document per app), the CRUD Editor edits its resources (rows in a table-shaped REST API).

Reading order

  1. Overview — what's a "resource table" and which services have them.
  2. API Contract — the REST shape the editor assumes.

At a glance

TargetsAny BLADE service that mounts the CRUD REST endpoints under /<app>/resources/api/
DiscoveryThe editor derives its API base URL from location.pathname; the same WAR can host the editor for any deployed app
Operationslist, get, put (create/update), delete; mutating operations support dry-run preview before commit
