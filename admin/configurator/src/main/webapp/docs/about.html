Where it fits

The Configurator is one of the BLADE admin-tier apps, hosted by the BLADE Admin Portal. The portal's left rail iframes this editor for in-context use; the standalone URL /blade/configurator/configurator.html opens the same editor directly, for direct access or bookmarking. (The bare /blade/configurator/ lands on this app's overview page, not the editor.)

Built on

  • BLADE framework — SettingsManager provides schema generation, MBean registration, and the on-disk layout.
  • Ace editor — JSON / Schema raw editor panels.
  • victools/jsonschema-generator — annotation-driven JSON Schema generation from Java POJOs.
  • WLS DomainRuntime MBean server — the federated MBean directory the Schema dropdown queries.

Reporting issues

File issues at the project tracker. Please include the AdminServer log around the time of the symptom, the target scope + schema name being edited, and (if possible) a screenshot.
