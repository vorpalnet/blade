BLADE Watcher

A backend-only file-system watcher. Monitors the BLADE configuration directory for changes and pushes updates into the matching SettingsMXBean. Has no editable UI — the watching happens automatically as soon as the WAR is deployed.

Legacy app

Watcher is kept in the OSS catalog for backward compatibility but is no longer the primary file-system watcher in modern deployments — that role moved into the Configurator's own ConfigurationMonitor. See the overview for when each is appropriate.
