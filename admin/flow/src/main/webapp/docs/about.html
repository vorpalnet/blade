Where it fits

The Flow Editor is one of the BLADE admin-tier apps, hosted by the BLADE Admin Portal. The portal's left rail iframes this editor for in-context use; the standalone URL /blade/flow/ works equivalently for direct access or bookmarking.

Built on

  • mxGraph (jgraphx 4.2.2) — canvas engine.
  • jQuery + jQuery UI — windows, dialogs, drag-and-drop.
  • FontAwesome 5 — palette and toolbar icons.
  • FsmarImportServlet / FsmarExportServlet — round-trip to the FSMAR descriptor.

Reporting issues

File issues at the project tracker. Please include the FSMAR generation (2 or 3), the descriptor that triggered the issue (sanitized of confidential names if applicable), and a screenshot or browser-console snippet.
