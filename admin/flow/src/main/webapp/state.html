For a State, this is the BLADE application it invokes. For an ingress (SBC, trunk, carrier), this is the entry state's name — a real FSMAR state with its own selectors and arrows. The default ingress is the null state (any source).
Set a distinct id to invoke the same application from more than one state (e.g. b2bua-caller and b2bua-callee, both running b2bua).
A when expression over the inbound request that routes its traffic to this ingress, e.g. ${originIP} insubnet '10.20.0.0/16'. Leave blank on the default ingress (it catches any source).
Run when a request enters this state. Each extracts a value into the routing context; transition conditions (when) and ${...} route templates read from it.
Route by plan (e.g. gold / silver / bronze): one transition per plan with mutually exclusive conditions, so evaluation order cannot matter.
