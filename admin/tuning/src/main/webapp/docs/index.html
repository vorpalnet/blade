Welcome

The BLADE Tuning dashboard is a curated view of the WLS / OCCAS performance knobs that BLADE operators most often need to change. Every knob maps to a WebLogic MBean attribute; the dashboard reads on page load and writes through the WLS edit session.

Reading order

  1. Overview — what the dashboard is curating and why.
  2. Sections Reference — every section, every knob.
  3. Edit Sessions — how WLS edit-and-activate works, and how the dashboard threads it.

At a glance

Reads fromDomainRuntime MBean server (federated, read-only)
Writes viaEdit MBean server (start-edit / activate cycle)
SectionsJVM, SIP timers, Work Manager, Cluster, Server tuning
Restart-requiredJVM and Initial JDBC capacity. Marked explicitly in each section.
Runtime-tunableSIP timers, Work Manager thresholds, most Server tuning. Take effect on Activate.
Quick action

Open the Dashboard, scroll to the section you want, change a value, press Save. Restart-required knobs are flagged in yellow.
