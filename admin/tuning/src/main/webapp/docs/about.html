Built on

  • WLS DomainRuntime + Edit MBean servers — reads and writes.
  • JAX-RS — per-section REST endpoints (one Java class per section: JvmSettings, SipTimerSettings, WorkManagerSettings, ClusterSettings, ServerTuningSettings).
  • Bootstrap 4 + jQuery — UI shell.

Reporting issues

File issues at the project tracker. Please include the WLS / OCCAS version, the failing section, and the AdminServer log lines around the time of the symptom.
